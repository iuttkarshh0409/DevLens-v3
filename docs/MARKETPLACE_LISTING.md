# DevLens GitHub Marketplace Listing Specifications

Marketing copy, branding requirements, and setup guidelines for DevLens V3 listing.

---

## 1. Product Description

### Short Description (Max 80 characters)
Automated repository health checks, quality audit scores, and pull request intelligence.

### Long Description
DevLens is a developer-native quality orchestration tool that automates repository audits.
It evaluates:
- **Quality Indicators**: Manifest files, configurations, testing suites, CI pipelines.
- **Pull Request Intelligence**: Publishes check runs and deterministic inline code recommendations.
- **Custom Configuration**: Tailor rules, caps, and scoring using versioned `.devlens.yml` policies.

---

## 2. Category & Branding
- **Primary Category**: Code quality
- **Secondary Category**: Security
- **Logo Color Spec**: HSL curated theme matching clean dashboard layout.
- **Product Icon**: Flat vector badge, minimum size 120px x 120px.

---

## 3. Marketplace Plans
- **Free Plan**: Unlimited analysis for public open-source repositories.
- **Developer Plan**: Complete check runs, PR review recommendations, and custom scoring policies for up to 5 private repositories.
- **Enterprise Plan**: Custom rules, organizational insights, priority support, and on-premises deployment support.
